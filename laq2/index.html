FEATURES

LAQ2の特徴

FEATURES 01
あっという間の
手間なしケア
よくわからないし、時間もかかりそう…と、面倒なイメージのある靴のケア。 LAQ2はこの“ケアの面倒さ”に向き合い、手間なしケアで靴のキレイを保つ欲張りなアイデアを形にしました。
FEATURES 02
簡単ケアでも
しっかり効果
LAQ2は、創業100年の老舗シューケアメーカーのノウハウを活かして生まれました。 長年の研究からつくられたアイテムだから、簡単ケアでもしっかりと効果を実感いただけます。
FEATURES 03
くらしに馴染む
デザイン
毎日気軽に使ってもらえるよう、見える場所にいつも置いておきたくなるようなデザインを目指しました。
